# Assets

## zylos-weixin.tgz

WeChat plugin for Zylos instances. Built from source at:
- Source: [wuxing/zylos-weixin](https://60.205.166.22:2222/wuxing/zylos-weixin) (Gitea)
- Submitted to Zylos official — link will be updated once open-sourced.

To rebuild from source:
```bash
cd /path/to/zylos-weixin
npm run build
npm pack
cp zylos-weixin-*.tgz /path/to/openclaw-hire/backend/assets/zylos-weixin.tgz
```
